PRACTICE PAPER - 01

Compulsory Section (Unit-V)
(a) Define an algorithm. What are its characteristics?
An algorithm is a logical, step-by-step procedure to solve a specific problem in a finite number of steps.
Characteristics: 1. Finiteness: Must terminate after a finite number of steps. 2. Definiteness: Each step must be clear and unambiguous. 3. Input: Zero or more inputs. 4. Output: At least one output. 5. Effectiveness: Operations must be basic and feasible.
(b) What is a flowchart? How does it differ from a decision table?
A flowchart is a pictorial or graphical representation of an algorithm using standard symbols (ovals, rectangles, diamonds) to show the flow of logic.
Difference: A flowchart visualizes the sequence of execution, whereas a Decision Table displays complex conditional logic in a tabular format, mapping conditions directly to actions without focusing on the flow.
(c) What are keywords in C? Give 10 examples.
Keywords are reserved words in C that have predefined meanings to the compiler. They cannot be used as variable names.
Examples: int, float, char, void, if, else, while, for, return, struct.
(d) Explain the difference between local and global variables.
Local Variables: Declared inside a function. Scope is limited to that function. Lifetime ends when the function exits.
Global Variables: Declared outside all functions. Scope is the entire program. Lifetime lasts for the full execution of the program.
(e) What is the purpose of the break statement? Where is it used?
The break statement is used to terminate a loop (for, while, do-while) or a switch statement immediately. Control is transferred to the statement immediately following the loop or switch.
(f) What is a union? How is it different from a structure?
A union is a user-defined data type where all members share the same memory location. The size of the union is equal to the size of its largest member.
Difference: In a structure, each member has its own separate memory, so the total size is (at least) the sum of all members. In a union, modifying one member affects the values of others.
(g) Explain the address operator (&) and indirection operator (*).
Address Operator (&): A unary operator that returns the memory address of a variable (e.g., &x gives the address of x).
Indirection Operator (*): Also called the dereference operator, it accesses the value stored at a specific address held by a pointer (e.g., *ptr gives the value at the address ptr).

PRACTICE PAPER - 02

Compulsory Section (Unit-V)
(a) What are data flow diagrams (DFD)?
A Data Flow Diagram (DFD) is a graphical representation of the "flow" of data through an information system. It models the process aspects of the system, showing sources, destinations, storage processes, and data paths. It is widely used in systems analysis.
(b) What are the different ways to pass parameters to a function?
1. Call by Value: A copy of the actual argument's value is passed. Changes in the function do not affect the original variable.
2. Call by Reference: The address (pointer) of the actual argument is passed. Changes in the function affect the original variable.
(c) What logical operators are available in C language?
1. Logical AND (&&): True only if both operands are true.
2. Logical OR (||): True if at least one operand is true.
3. Logical NOT (!): Reverses the logical state of its operand.
(d) What is the significance of static storage class?
A static variable preserves its value between function calls. It is initialized only once (default is 0) and exists for the lifetime of the program, even though its scope is local to the block where it is defined.
(e) What is pseudo code?
Pseudo Code is an informal high-level description of the operating principle of a computer program. It uses natural language structure rather than programming language syntax, making it easier for humans to understand the logic before coding.
(f) How are structures different from arrays?
Arrays hold a collection of elements of the same data type (homogeneous).
Structures hold a collection of elements of different data types (heterogeneous) grouped together under a single name.
(g) Explain what happens during compilation and execution of a C program.
Compilation: The compiler translates the human-readable C source code into machine-readable object code. It checks for syntax errors.
Execution: The operating system loads the linked executable file into memory and the CPU executes the instructions to perform the defined tasks.

PRACTICE PAPER - 03

Compulsory Section (Unit-V)
(a) What is the use of the goto statement in a program?
The goto statement provides an unconditional jump from the goto to a labeled statement in the same function.
Disadvantage: It makes logic hard to follow ("spaghetti code") and makes debugging difficult.
(b) What is debugging? What are the different types of errors?
Debugging is the process of finding and resolving defects or problems within the program that prevent correct operation.
Types of Errors:
1. Syntax Errors: Grammar rules violated (e.g., missing semicolon).
2. Logical Errors: Wrong logic, correct syntax (e.g., wrong formula).
3. Runtime Errors: Crashes during execution (e.g., divide by zero).
4. Linker Errors: Missing functions or libraries.
(c) Define escape sequences in C. Give 5 examples.
Escape Sequences are characters that consist of a backslash (\) followed by a letter or combination of digits. They represent non-printable characters.
Examples: \n (newline), \t (horizontal tab), \b (backspace), \" (double quote), \\ (backslash).
(d) What is a function prototype? Why is it important?
A function prototype is a declaration of a function that specifies the function's name, return type, and parameters, but not the body.
It informs the compiler about the function interface before it is defined, allowing calls to the function even if the definition comes later.
(e) What is the use of static variable?
As mentioned, a static variable retains its value between multiple function calls. It's useful for counting how many times a function is called or maintaining state without using global variables.
(f) How are structures different from unions?
(See Paper 1 answer). Structures allocate memory for all members (sum of sizes), while Unions allocate memory only for the largest member (shared memory).
(g) What is the purpose of the header file stdio.h? List 5 functions.
stdio.h (Standard Input Output) is a header file containing definitions of macros, constants, and declarations of functions used for input and output operations.
Functions: printf(), scanf(), gets(), puts(), fopen().

PRACTICE PAPER - 04

Compulsory Section (Unit-V)
(a) What is a pseudo code?
An artificial and informal language that helps programmers develop algorithms without worrying about specific syntax.
(b) What is the character set of C?
The C character set includes:
1. Alphabets: A-Z, a-z.
2. Digits: 0-9.
3. Special Characters: ~ ! @ # % ^ & * ( ) _ - + = { } [ ] \ | ; : " ' <> , . ? /
4. White Spaces: Space, Tab, Newline.
(c) Differentiate between break and continue statements.
Break: Terminates the loop completely and control moves to the next statement after the loop.
Continue: Skips the current iteration of the loop and passes control to the next iteration check.
(d) What are storage classes in C? Name all 4.
Storage classes determine the scope, visibility, and lifetime of a variable.
The 4 types are: 1. auto (default for locals) 2. register (stored in CPU register) 3. static (persists value) 4. extern (global visibility)
(e) What is the use of conio.h header file?
conio.h (Console Input Output) is a specific header (mostly MS-DOS/Windows) used for console I/O. Common functions found in it are clrscr() (clear screen) and getch() (get character without echo).
(f) How are arrays of structures used?
We can create an array where each element is a structure. This is very useful for databases, like storing records of 50 students.
Example: struct Student batch[50]; creates 50 instances of the Student structure.
(g) What is the significance of free() function in dynamic memory allocation?
The free() function is used to deallocate (release) the memory that was previously allocated dynamically using malloc, calloc, or realloc. Faisling to use it leads to memory leaks, where memory is consumed but never returned to the system.

PRACTICE PAPER - 05

Compulsory Section (Unit-V)
(a) Explain keywords and constants.
Keywords: Reserved words with fixed meanings (e.g., if, while).
Constants: Fixed values that do not change during program execution (e.g., 3.14, 'A', 100).
(b) What is the significance of the address operator (&) and indirection operator (*)?
(See Paper 1). & gets the address of a variable. * gets the value at an address (pointer).
(c) What is a union? How is it different from a structure?
(See Paper 1). Union uses shared memory for members; Structure uses separate memory.
(d) What is the use of break statement in a program?
(See Paper 1/4). Used to exit a loop or switch structure instantly.
(e) What is binary search?
Binary Search is an efficient algorithm for searching an element in a sorted array. It works by repeatedly dividing the search interval in half. If the value is less than the middle element, it searches the left half; otherwise, the right half. Time complexity is O(log n).
(f) How does dynamic memory allocation differ from static memory allocation?
Static Allocation: Variables are allocated memory at compile time (stack/data segment). Size is fixed.
Dynamic Allocation: Memory is allocated at runtime (heap) using functions like malloc(). Size can change, and it must be manually freed.
(g) What is the significance of null terminator (\0) in strings?
The null terminator ('\0') marks the end of a character string in C. Since C strings are just arrays of characters, functions like printf or strlen rely on this special character to know where the string data stops.
